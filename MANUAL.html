CellAppCP3 · v2.3.0

荧光细胞
分割与亮度排名

CellAppCP3 是基于 Cellpose cyto3 模型的命令行工具，用于分割荧光细胞图像并按亮度排序。

Cellpose cyto3 · Python CLI
快速开始

三步完成识别

Step 01

放入图片

把需要识别的图片放在 input 文件夹。

input/ → 你的图片
Step 02

运行程序

在终端输入命令运行脚本。

python cells_find.py
Step 03

查看结果

查看 results 文件夹中的识别结果。

results/ → 识别结果
目录结构

各文件夹是什么

程序运行前，先了解每个文件夹和文件的作用。
input/
把要识别的图片放进去。程序启动后会自动扫描此文件夹里所有图片。
results/
存放识别结果的地方。每张图片对应一个子文件夹，识别完自动生成。
cyto3
程序识别细胞用的智能模型文件，不要移动或删除它。
settings.txt
所有参数的配置文件。直接用文本编辑器打开修改，保存后下次运行生效。
参数说明

settings.txt 配置

直接编辑 settings.txt 修改下方参数，保存后重新运行即可生效，无需改代码。
识别模型参数
参数默认值说明建议范围
diameter120细胞直径（像素），越小识别更小的细胞50–200
flow_threshold0.95流场一致性，越低检测越宽松0.1–1.0
cellprob_threshold1.0细胞概率门槛，越低检测越多-6.0–6.0
min_size200最小细胞面积（像素），排除过小区域50–500
resample0是否重采样，1=开启（更慢更准）0 或 1
niter200迭代次数，越少越快50–400
过滤参数
参数默认值说明建议范围
hull_comp0.85凸包完整度，越低越宽松（过滤残缺细胞）0.6–0.95
circle_comp0.65圆形完整度，越低越宽松0.4–0.85
dark_threshold15暗区过滤，低于此均值的区域跳过5–30
area_ratio0.15面积过滤，小于中位数此比例的细胞剔除0.05–0.3
circularity0.5圆形度，越低接受越不圆的形状0.3–0.7
min_pixels50亮度计算最少像素数20–100
brightness_top_pct0.05亮度取前百分比0.01–0.1

参数默认值说明建议范围
large_mode1大图处理方式，1=自动分块，0=整图0 或 1
tile_w2048分块宽度（像素）1024–4096
tile_h1080分块高度（像素）512–2160
tile_overlap0.2分块重叠比例0.1–0.3

参数默认值说明建议范围

输出文件

结果在哪里

每张图片识别完成后，在 results/ 文件夹里生成一个以图片名命名的子文件夹。

举例：如果你放入的图片叫 sample.jpg，识别完成后会生成：

results/
  └── sample/
      ├── sample_result.png   ← 标注结果图，Top N 绿色高亮
      ├── sample_raw_detections.png   ← 过滤前所有原始检测点
      ├── sample_top3.csv   ← Top N 细胞坐标和亮度
      ├── sample_data.csv   ← 全部有效细胞数据
      └── crop/   ← 子文件夹，存放裁剪图
            ├── top1.png
            ├── top2.png
            └── top3.png
文件说明
sample_result.png标注后的结果图，Top N 细胞绿色轮廓，其余黄色
sample_raw_detections.png过滤前的诊断图，显示所有原始检测点
sample_top3.csv最亮前 N 个细胞的坐标与亮度值
sample_data.csv所有有效细胞的编号、直径、坐标、亮度
crop/top1.png …Top N 细胞的放大裁剪图，保存在 crop 子文件夹里
常见问题

遇到问题怎么办

细胞完全没被检测到怎么办
降低 cellprob_threshold 到 0 或 -2 试试。这个值越低，程序越容易把区域认定为细胞。
检测到太多不像细胞的噪点怎么办
提高 hull_comp 和 circle_comp 参数值。这两个值越高，形状不够圆整的区域会被过滤掉。
程序运行很慢怎么办
降低 niter 参数（比如改为 100），同时把 resample 设为 0。
大图片导致程序因内存不足而崩溃怎么办
确认 large_mode=1，并减小 tile_w 和 tile_h，让程序分更小的块来处理，降低每次占用的内存。
描边轮廓太细看不清怎么办
调大 contour_thickness 参数，比如改为 3 或 4，描边线会变粗。
裁剪图片在哪里找
裁剪图保存在 results/图片名/crop/ 文件夹里，文件名为 top1.png、top2.png 等。
