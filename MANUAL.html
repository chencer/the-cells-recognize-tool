CellAppCP3 使用手册

CellAppCP3 是基于 Cellpose cyto3 模型的命令行界面工具，用于分割荧光细胞图像并按亮度排名，支持大图分块识别与 settings.txt 参数热配置。

快速开始

三步完成细胞图像识别，无需任何代码改动。

1
放入图片 将荧光细胞图像（.tif / .png / .jpg）复制到 input/ 文件夹中。
2
运行脚本 在项目根目录打开终端，执行： 
python cells_find.py
3
查看结果 结果自动生成到 results/{图片名}/ 目录，包含标注图、CSV 数据和 Top 细胞裁剪图。

settings.txt 参数说明

修改 settings.txt 后保存，下次运行自动读取，无需改代码。未找到文件时使用内置默认值。

🧠 模型参数
参数默认值说明建议范围
diameter120细胞直径（像素），根据实际细胞大小调整50–200
flow_threshold0.95流场一致性阈值，越低越宽松，检测更多候选0.1–1.0
cellprob_threshold1.0细胞概率阈值，越低检测越多，负值大幅放宽-6.0–6.0
min_size200最小细胞面积（像素），排除过小的噪点区域50–500
resample0是否对输出进行重采样，1=开启（更准但更慢）0 或 1
niter200流场迭代次数，减少可显著提速50–400
🔬 过滤参数
参数默认值说明建议范围
hull_comp0.85凸包完整度阈值，低于此值的不规则区域被过滤0.6–0.95
circle_comp0.65圆形完整度阈值，过滤边缘残缺的细胞0.4–0.85
dark_threshold15暗区过滤，均值亮度低于此值的区域跳过5–30
area_ratio0.15面积过滤，小于中位数此比例的细胞被剔除0.05–0.3
circularity0.5圆形度阈值，越低接受越不圆的形状0.3–0.7
min_pixels50亮度计算所需最少像素数，低于此数跳过20–100
brightness_top_pct0.05亮度取 top 百分比均值，0.05 表示前 5%0.01–0.1
🗂 分块参数
参数默认值说明建议范围
large_mode1大图处理方式，1=自动分块识别，0=整图识别0 或 1
tile_w2048分块宽度（像素）1024–4096
tile_h1080分块高度（像素）512–2160
tile_overlap0.2分块重叠比例，防止边界细胞被截断0.1–0.3
🎨 输出参数
参数默认值说明建议范围
top_n3高亮显示并导出最亮的前 N 个细胞1–20
crop_pad2.0Top 裁剪图范围（相对细胞半径的倍数）1.5–3.0
contour_thickness2轮廓描边线宽（像素）1–5
font_scale0.7标注文字大小倍数0.3–1.5

输出文件说明

每张图片处理完成后，在 results/{图片名}/ 目录下生成以下文件：

🖼
{图片名}_result.png 标注结果图。Top N 细胞绿色轮廓高亮，其余细胞黄色标注编号，80% 内圈压暗显示采样区域。
🔍
{图片名}_raw_detections.png 诊断图。显示过滤前的所有原始检测点（绿色圆点），用于调参参考。
📊
{图片名}_top{n}.csv Top N 细胞数据，含排名、坐标 X/Y、亮度值。
📋
{图片名}_data.csv 全部有效细胞数据，含编号、直径（px）、坐标 X/Y、亮度值。
✂️
top1.png / top2.png / ... Top N 细胞的放大裁剪图，左上角标注排名，方便直接查看最亮细胞形态。

常见问题

细胞没被检测到，结果为空
降低 cellprob_threshold，默认值 1.0 较严格。可先尝试改为 0.0，若仍无结果改为 -2.0 甚至 -4.0。同时检查 diameter 是否与实际细胞大小匹配。
检测到太多噪点或不规则区域
提高 hull_comp（如改为 0.90）和 circle_comp（如改为 0.75），这两个阈值控制形状完整度，越高过滤越严格。也可提高 circularity 进一步限制形状圆度。
运行速度太慢
降低 niter（如改为 100 或 50）可显著提速，影响精度有限。确保 resample=0（关闭重采样）。大图时确认 large_mode=1 以启用分块处理，避免整图加载 OOM。
大图处理崩溃或内存不足
确认 large_mode=1。若仍崩溃，减小分块尺寸：将 tile_w 改为 1536，tile_h 改为 768。在 Apple Silicon 上 MPS 显存不足时会自动清理缓存，如持续崩溃可尝试降低分块尺寸。
裁剪图太小，看不清细胞
增大 crop_pad，默认值 2.0 表示裁剪范围为细胞半径的 2 倍。改为 3.0 或 4.0 可获得更大的上下文区域。
暗细胞被过滤掉了
降低 dark_threshold（如改为 5），该参数过滤平均亮度低于阈值的暗区，降低后更多暗区域会被保留参与识别。
