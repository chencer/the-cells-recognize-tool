# The Cells Recognize Tool

基于 **Cellpose cyto3** 模型的荧光细胞图像分割与亮度排名桌面工具。当前版本：**v2.1.0**

---

## 窗口布局

```
┌──────────────────────────────┬──────────────┐
│                              │  细胞数据     │
│       图像预览区              │  排名 X Y D 亮│
│   (左侧，自适应缩放)           │  #1 ...      │
│                              │  #2 ...      │
│                              │  ...（可滚动）│
└──────────────────────────────┴──────────────┘
```

## 功能概览

| 功能 | 说明 |
|------|------|
| 细胞分割 | Cellpose cyto3 模型，一键完成荧光图像分割 |
| 细胞总数统计 | 状态栏实时显示检测总数 / 过滤后有效数 |
| 边缘完整度过滤 | 剔除与图像边界接触的截断细胞（≥ 90%） |
| 面积 + 圆形度过滤 | 去除噪声点（面积阈值 15%）和异形物体（圆形度 ≥ 0.5）|
| 亮度排名 | 80% 内圈前 5% 最亮像素均值，避免大细胞虚排 |
| 外圈遮罩可视化 | 所有有效细胞外圈 20% 压暗，直观展示亮度采样范围 |
| Top 3 标注 | 绿色轮廓 + `序号 \| 亮度值`，其余细胞黄色轮廓 + 白色编号 |
| 右侧数据面板 | 固定 280px 面板，列出所有有效细胞排名/坐标/直径/亮度 |
| 原图 / 结果图切换 | 一键对比，数据面板保持不动 |
| CSV 数据导出 | 一键保存结果图 + top3.csv + data.csv |
| 异步加载 | 启动时后台加载模型，进度动画实时反馈 |
| UI 风格 | OLED dark cinema + glassmorphism，Windows 原生标题栏，固定大小 |

## 使用方法

**Windows 用户**（推荐）：

1. 从 [Releases](https://github.com/chencer/the-cells-recognize-tool/releases) 下载最新 `.exe`
2. 双击运行，等待模型加载完成（首次约 10–30 秒）
3. 点击 **导入图片** 选择荧光图像（.tif / .png / .jpg）
4. 点击 **细胞识别** 开始分割
5. 右侧面板自动填充所有有效细胞数据（Top 3 绿色高亮）
6. 点击 **保存** 导出 CSV 和结果图到 `细胞数据/` 目录

**开发运行**：

```bash
pip install -r requirements.txt
python cells_find.py
```

## 过滤算法说明

### 边缘完整度过滤

检测细胞 mask 的任意像素是否触碰图像四条边。接触则判定为边缘截断细胞并剔除。

> 注：凸包比方法无法识别圆弧截断（弧形 cap 本身是凸形，比值恒 ≈ 1.0），因此使用边界像素接触检测。

### 面积 + 圆形度过滤

- **面积过滤**：中位数面积 × 15%，去除极小噪声点
- **圆形度** = 4π × 面积 / 周长²，低于 0.5 的异形物体剔除

### 亮度得分

取细胞 80% 内圈区域内**前 5% 最亮像素的均值**作为亮度得分，精确识别小而亮的目标。
